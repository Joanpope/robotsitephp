Admin

Extends \Models\Business\User

Classe Admin que hereta de User i que te privilegis per realitzar operacions CRUD sobre qualsevol model.

S'utilitza per gestionar els usuaris de l'aplicació per ser posteriorment distribuïts en: administradors o treballadors
