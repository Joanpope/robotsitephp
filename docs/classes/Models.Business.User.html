User

Extends \Models\Business\DataObject

Classe abstracta User, hereta de DataObject i permet gestionar el CRUD de la taula workers a partir dels seus metodes i atributs.

S'utilitza per gestionar els usuaris de l'aplicació per ser posteriorment distribuïts en: administradors o treballadors
